# spy-cat-test

## Technologies
- Django Rest Framework

## Commands to run a project
- Firstly, open cmd inside src/ folder.
- Then run this command to apply migrations:
```
python manage.py migrate
```

- Secondly, run dev server with:
```
python manage.py runserver
```
